## Featured Projects

### 🚇 CMMT — Chennai Multi-Modal Transit Planner
`Data Engineering · Personal Project · 2025`

End-to-end transit planning system built on real **GTFS datasets** spanning MTC buses, Chennai Metro, and suburban rail. Solves the multi-modal route optimisation problem with a Pareto-front approach — not just fastest, but best across *all* dimensions simultaneously.

- Ingested and normalised raw GTFS feeds across 3 networks
- Constructed a weighted multi-modal graph with **NetworkX**
- Built a **multi-objective Pareto scoring engine** ranking routes by time, cost, distance, and CO₂
- Full pipeline: raw data → graph construction → route ranking → interpretable output

`Python` `NetworkX` `GTFS` `Pareto Optimisation` `Graph Algorithms`

---

### 🧠 ASD Early Detection — Clinical ML System
`Healthcare ML · Academic Project · 2024`  
🔗 [View Repository](https://github.com/srighanesh-sriv/asd-detection-ml)

Predictive ML pipeline for early Autism Spectrum Disorder detection in children under the age of 2 years. Granted **ethical clearance from SRM Medical Council** (valid till 2027)

- Applied **PCA** for dimensionality reduction on a 21-feature behavioural dataset (704 instances)
- Benchmarked **8 models**: Random Forest, SVM, Neural Networks, KNN, LightGBM, Decision Tree, t-SNE, K-Means
- Result: **PCA + Random Forest → 99% accuracy, AUC 0.99**
- Deployed a **prototype clinical web interface** for real-time screening and therapy recommendation

`Python` `Scikit-learn` `PCA` `Flask` `Seaborn` `Plotly`

### 🔍 Fingerprint Recognition System
`Computer Vision · Academic Project · 2023`

Biometric identification pipeline using minutiae-based feature extraction with Python and OpenCV. Designed for reliable matching under **degraded image conditions** — noisy scans, partial prints, and low resolution.

- Full preprocessing pipeline: normalisation, binarisation, thinning
- Minutiae detection and matching under noisy conditions
- Systematic analysis of data quality's impact on biometric accuracy

`Python` `OpenCV` `Computer Vision` `Feature Engineering`
